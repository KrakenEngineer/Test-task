Открывать на версии Unity 6.3 LTS (6000.3.4f1)

Уточнения и небольшие доработки:
1) Патруль ходит по прямой, между крайними клетками комнаты. Разворачивается если подходит к краю комнаты или препятствию, фактически может быть заперт
2) Огненные шары разрушаются об препятствия, фактически могут быть заблокированы (необходимо для прохождения уровня)
3) Игрок при построении маршрута может двигаться по вертикали, горизонтали, диагонали
4) Игрок при построении маршрута не учитывает огненные шары и патруль, но динамически меняет его если встречает на пути препятствие
5) Игрок не ходит по диагонали вплотную к препятствию, поэтому не может преодолеть диагональную стену
6) Препятствие не может быть установлено на месте игрока, патруля, огненного шара или рядом с ними, но всегда может быть убрано
7) Нажатием на R можно перезапустить игру

Спрайты:
1) Игрок - викинг в правой части экрана
2) Пушки - стенки слева сверху от игрока
3) Патруль - скелет с красными рогами
4) Цель - звезда в противоположном углу комнаты
5) Препятствие - камень

Прилагается сборка под windows
